Components

Lullaby is a design system created to showcase the capabilities of UI Composer.

The Editor Pattern

In UI Composer, components are functorial. In short, a component is something which visualizes and/or edits some datum T. Because data types are composable, components should be composable, too.

Views and Inputs

TextEdit inputs can edit a String.

A ValidatedTextEdit can edit any impl FromStr.

Actions and Effects

Buttons are the UI equivalent of Effects. An Effect represents a change in state, whether that's a property of an element, dispatching a task, or incrementing a number.
